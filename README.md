# Husarion Camera Node Project
This repository contains the ROS implementation for the camera interface on the Husarion robot.

## Installation
1. Clone this repository into your `catkin_ws/src` folder.
2. Run `catkin_make` in the workspace root.
3. Source the workspace: `source devel/setup.bash`

## Running the Node
To start the node, use:
`rosrun tutorial_pkg tutorial_pkg_node`
